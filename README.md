## 📊 AdventureWorks Business Intelligence Dashboard

### 🛠 Tools Used

* Power BI Desktop
* Excel (Data Cleaning & Preparation)

---

### 📌 Project Overview

During this project, I built an interactive Power BI dashboard to analyze sales performance across products, customers, and regions using the AdventureWorks dataset.
The goal was to create a clear and user-friendly view that helps in understanding business performance and identifying key trends.

---

### 📈 Key Insights

* Revenue shows a consistent upward trend over time, indicating overall business growth
* Tires and Tubes category generates the highest number of orders
* Some products have relatively higher return rates, which may highlight potential quality or customer satisfaction issues
* Sales performance varies across regions, with certain markets contributing more significantly to total revenue

---

### 🚀 Business Value

This dashboard helps stakeholders to:

* Track key KPIs such as Revenue, Profit, Orders, and Return Rate
* Identify top-performing products and customers
* Monitor trends over time for better decision-making
* Explore data dynamically using filters and slicers

---

### 🔍 Key Features

* KPI cards for high-level performance tracking
* Time-series analysis for revenue trends
* Geographic visualization of sales distribution
* Customer segmentation and top customer analysis
* Advanced visuals such as Key Influencers

---

### 💡 What I Learned

This project helped me improve my skills in Power BI, especially in building interactive dashboards and turning raw data into meaningful insights.
